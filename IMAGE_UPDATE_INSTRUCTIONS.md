# Updating Product Images Using Centralized Image Configuration System

## Overview
This document outlines the steps to update product images for our products using the centralized image configuration system without the need to edit HTML files directly.

## Steps to Update Product Images:

1. **Access the Centralized Image Configuration System**  
   - Navigate to the centralized image configuration interface on your web browser.
   - Log in with your credentials if required.

2. **Select the Product**  
   - In the dashboard, browse or search for the product whose image you want to update.
   - Click on the product name to access its details.

3. **Upload New Image**  
   - Look for the image upload section within the product detail page.
   - Click on the "Upload Image" button.
   - Choose the new image file from your computer. Make sure the file format is supported (JPEG, PNG, etc.).
   - After selecting the file, confirm the upload by clicking on the submit or upload button.

4. **Preview the Image**  
   - Once uploaded, a preview of the image should appear.
   - Ensure that the image is appropriate and meets the quality standards.

5. **Save Changes**  
   - After confirming the new image, look for the "Save Changes" or "Update Product" button.
   - Click on it to apply the changes.

6. **Verify the Update**  
   - Navigate back to the product page and refresh to see the updated image.
   - Ensure the new image displays correctly on the site.

## Important Notes:
- Always use high-quality images to enhance the visual appeal of the products.
- Keep backups of previous images if needed for future reference.
- If you encounter any issues, reach out to the support team for assistance.

By following these instructions, you can successfully update product images in the centralized image configuration system without modifying any HTML code directly.
